# 🐾 PetCare - Clínica Veterinária

![React](https://img.shields.io/badge/React-18.3.1-61dafb?style=for-the-badge&logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-3178c6?style=for-the-badge&logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4.1-38bdf8?style=for-the-badge&logo=tailwind-css)

Um projeto moderno e responsivo de landing page para clínica veterinária, desenvolvido com as melhores práticas de desenvolvimento web.

## 🚀 Tecnologias Utilizadas

- **React 18.3.1** - Biblioteca JavaScript para construção de interfaces
- **TypeScript** - Superset JavaScript que adiciona tipagem estática
- **Tailwind CSS 4.1** - Framework CSS utility-first
- **Motion (Framer Motion)** - Biblioteca para animações fluidas
- **Lucide React** - Ícones modernos e customizáveis
- **Vite** - Build tool extremamente rápido

## ✨ Funcionalidades

- ✅ Design responsivo para todos os dispositivos
- ✅ Animações suaves e interativas
- ✅ Navegação por seções com scroll suave
- ✅ Menu mobile com animações
- ✅ Formulário de contato funcional
- ✅ Sistema de design com cores verde e rosa
- ✅ Componentes reutilizáveis e bem estruturados
- ✅ Código TypeScript totalmente tipado
- ✅ Performance otimizada

## 🎨 Seções do Site

1. **Hero** - Apresentação principal com call-to-action
2. **Serviços** - Catálogo completo de serviços veterinários
3. **Equipe** - Apresentação dos profissionais
4. **Depoimentos** - Feedback de clientes satisfeitos
5. **Contato** - Formulário e informações de contato
6. **Footer** - Links úteis e redes sociais

## 📦 Instalação

```bash
# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## 🏗️ Estrutura do Projeto

```
src/
├── app/
│   ├── components/
│   │   ├── ui/           # Componentes de UI reutilizáveis
│   │   ├── Header.tsx    # Cabeçalho e navegação
│   │   ├── Hero.tsx      # Seção principal
│   │   ├── Services.tsx  # Seção de serviços
│   │   ├── Team.tsx      # Equipe de profissionais
│   │   ├── Testimonials.tsx # Depoimentos
│   │   ├── Contact.tsx   # Formulário de contato
│   │   └── Footer.tsx    # Rodapé
│   └── App.tsx           # Componente principal
└── styles/
    ├── theme.css         # Variáveis de tema customizadas
    └── tailwind.css      # Configurações do Tailwind
```

## 🎨 Sistema de Cores

O projeto utiliza uma paleta de cores profissional que combina verde (representando natureza e saúde) com rosa (representando cuidado e amor):

- **Primary (Verde)**: `#2d5f3f` - Usado para elementos principais
- **Secondary (Rosa)**: `#f4a5c0` - Usado para destaques
- **Accent (Rosa Claro)**: `#ffd4e5` - Usado para backgrounds suaves

## 📱 Responsividade
